1) Запускаем Neo4j
2) Снизу рядом с заголовком "File" есть кнопочка [Reveal files in File Explorer]
3) Кликаем, вставляем файл из папки db проекта в эту папку
4) Под заголовком "File" появится этот файл
5) Нажать "..." -> "Create a new DBMS from dump
6) Вводите название проекта, пароль
7) Создастся ваш проект
8) Нажать Open проекта
9) вводим запрос match (n) return n и получем текущий граф
